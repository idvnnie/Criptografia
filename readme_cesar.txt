Cifrado César en C
Este programa usa el Cifrado César, el cual es un método de cifrado por sustitución en el que cada letra del mensaje se desplaza un número fijo de posiciones en el alfabeto.

FUNCIONAMIENTO
Al ejecutar el programa, se pueden elegir entre tres opciones:

Cifrar un mensaje: Se introduce un texto en minúsculas y una clave de tipo numérica. después, el programa aplica el desplazamiento y muestra el mensaje cifrado.
Descifrar un mensaje: Se introduce un mensaje cifrado y la misma clave utilizada en el cifrado. El programa revierte el proceso y muestra el mensaje original.
Salir: Termina la ejecución del programa.

Requerimientos
Es necesario contar con un compilador de C, y ejecutar código en una consola/terminal

COMPILADO
gcc cifradocesar.c -o cifradocesar
EJECUCIÓN
./cifradocesar


IMPORTANTE
El programa solo aCEPTA letras minúsculas y la clave debe ser un número entero.
